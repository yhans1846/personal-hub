# 新增功能：沉浸式创作模式（Focus Mode）

## 目标

为 Personal Hub 的笔记模块增加一个「沉浸式创作模式」，提升长时间写作体验。

参考产品：

- Notion
- Obsidian
- Typora
- GitBook
- Craft

整体风格保持 Personal Hub 当前设计语言（Apple + Linear），简洁、不浮夸。

---

# 功能一：全屏创作

在编辑器右上角新增按钮：

```
□ 全屏
```

点击后：

- 编辑器进入浏览器全屏（Fullscreen API）
- 左侧菜单隐藏
- 顶部 Header 隐藏
- 面包屑隐藏
- 内容区域占满整个屏幕
- 页面背景保持浅灰色
- 编辑区域最大宽度 900~1000px，居中显示

页面布局：

```
┌──────────────────────────────────────────────┐
│                                              │
│              笔记标题                         │
│                                              │
│──────────────────────────────────────────────│
│                                              │
│                                              │
│                                              │
│              Markdown 编辑器                 │
│                                              │
│                                              │
│                                              │
│                                              │
│                                              │
└──────────────────────────────────────────────┘
```

ESC 或点击退出按钮恢复正常布局。

---

# 功能二：实时预览

编辑器右上角新增：

```
预览

实时预览

全屏
```

其中：

实时预览采用 Switch 开关。

开启后：

页面切换为左右布局。

```
┌──────────────────────────────────────────────┐
│ Markdown 编辑 │        Preview              │
├──────────────────────────────────────────────┤
│               │                             │
│               │                             │
│               │                             │
│               │                             │
│               │                             │
│               │                             │
└──────────────────────────────────────────────┘
```

比例：

50 : 50

可拖拽调整宽度：

40 : 60

60 : 40

默认：

50 : 50

---

# 功能三：预览滚动同步

编辑器滚动：

右侧 Preview 同步滚动。

Preview 滚动：

左侧编辑器同步滚动。

保持阅读位置一致。

---

# 功能四：阅读模式

关闭工具栏。

仅显示：

标题

正文

Preview

适合校对文章。

---

# 功能五：Focus Mode

新增：

```
Focus Mode
```

开启后：

隐藏：

- 左侧菜单
- Header
- Footer
- Breadcrumb
- Toolbar（可自动隐藏）

仅保留：

标题

正文

底部状态栏

整个页面更加沉浸。

---

# 功能六：底部状态栏

编辑器底部增加状态栏。

实时显示：

```
1200 字

12 段

阅读时间 5 分钟

最后保存：14:32

Markdown

UTF-8
```

自动更新。

---

# 功能七：自动保存

编辑内容发生变化：

3 秒自动保存。

状态：

```
正在保存...

✓ 已保存

最后保存 14:32
```

无需点击保存按钮。

---

# 功能八：快捷键

Ctrl + S

立即保存。

Ctrl + Shift + F

进入全屏。

Ctrl + Shift + P

开启实时预览。

ESC

退出全屏。

---

# 功能九：主题

保持当前 Personal Hub 风格。

不要：

- 毛玻璃
- 发光
- 渐变背景
- 炫酷动画

要求：

- 极简
- 留白充足
- 编辑区域最大宽度 900~1000px
- 字体舒适
- 标题突出
- Preview 与编辑器保持一致样式

参考：

Notion Editor

Typora

Craft

---

# 功能十：状态记忆

记录用户上次使用状态：

- 是否开启实时预览
- 是否全屏
- 编辑器宽度比例
- Focus Mode 是否开启

再次进入编辑器时自动恢复。

---

## 实现要求

前端：

- Vue3
- Element Plus
- Pinia
- Fullscreen API
- Splitpanes（左右拖拽）
- localStorage 保存用户偏好

要求：

- 不影响现有编辑功能
- 保持现有 API 不变
- 新功能可独立开关
- 代码符合项目现有架构和编码规范
