# Personal Hub - 项目介绍

## 项目定位

面向个人使用的学习与知识管理系统。前后端分离，围绕个人学习场景设计。

## 开发目标

构建个人知识管理平台，练习 Spring Boot 3 + Vue 3 前后端分离开发，项目长期维护，功能围绕个人场景。

## 系统架构

```
Vue 3 + Vite + TypeScript → Axios → RESTful API → Spring Boot 3 → MyBatis-Plus → MySQL
```

## 项目目录

```
personal-hub
├── backend/       # Spring Boot 后端
├── frontend/      # Vue 3 前端
├── sql/           # 数据库脚本
├── docs/          # 项目文档
└── README.md
```

## 功能规划

### 第一阶段（MVP）
- **用户模块**: 登录、JWT 认证、修改密码、个人信息
- **Markdown 笔记**: 新建/编辑/预览/自动保存、分类/标签、搜索/收藏/回收站
- **学习记录**: 日期/主题/时长/内容/心得 CRUD
- **Todo**: CRUD、完成状态、优先级、截止日期
- **文件管理**: 上传/下载/删除/分类/搜索（PDF/Word/Excel/PPT/MD/图片/ZIP）

### 第二阶段
- [x] **日记**: Markdown 编辑，月/周聚合，搜索
- [x] **收藏夹**: 网址 CRUD，分类/标签，搜索
- [x] **学习计划**: CRUD，进度跟踪，关联学习记录
- [x] **阅读记录**: CRUD，章节进度，阅读笔记

### 第三阶段
- [x] **标签系统**: 统一管理，全模块关联，筛选统计（ph-tag 模块，替换 note_tag + bookmark_tags）
- [x] **Dashboard**: 系统概览（学习时长/待办/最近笔记/统计卡片，ph-dashboard 模块）
- [x] **数据统计**: 学习/笔记/Todo/阅读趋势（ECharts 折线图，ph-dashboard 模块）
- **全局搜索**: 跨模块统一搜索、高亮、历史

### 后续可扩展
AI 总结/问答 / OCR / PDF 预览 / MD 导出 PDF/HTML / 数据备份 / WebDAV / Docker / 深色/多主题

## 模块依赖关系

```
用户认证（前置）
    ├── 笔记 ── 标签系统
    ├── 学习记录 ── 数据统计
    ├── Todo
    ├── 文件管理
    ├── 日记 ── 标签系统
    ├── 收藏夹 ── 标签系统
    ├── 学习计划
    └── 阅读记录
            ↓
      Dashboard（聚合）+ 全局搜索（跨模块）
```

## 项目原则

1. 可读性优先 2. 可维护性优先 3. 结构统一 4. 不重复代码 5. 优先复用
6. 不随意引入依赖 7. 前后端命名一致 8. 接口风格统一 9. 数据库设计一致 10. 新增功能更新文档
