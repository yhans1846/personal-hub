# Personal Hub - 项目介绍

## 项目定位

**Personal Hub** 是一个面向个人使用的学习与知识管理系统。项目采用前后端分离架构，围绕个人学习场景设计，帮助用户建立属于自己的知识管理平台。

## 开发目标

- 构建一个属于自己的学习与知识管理平台
- 练习 Spring Boot 3 与 Vue 3 前后端分离开发
- 熟悉现代 Java Web 开发流程
- 项目长期维护，持续增加功能
- 所有功能均围绕个人使用场景设计

## 系统架构

```
Vue 3 + Vite + TypeScript
        │
      Axios
        │
  RESTful API（JSON）
        │
  Spring Boot 3
        │
   Spring MVC
        │
  MyBatis-Plus

      MySQL
```

## 功能规划

### 第一阶段（MVP）

| 模块          | 功能                                             |
| ------------- | ------------------------------------------------ |
| 用户模块      | 登录、JWT 身份认证、修改密码、个人信息           |
| Markdown 笔记 | 新建/编辑笔记、实时预览、自动保存、分类/标签管理、搜索、收藏、最近编辑、回收站 |
| 学习记录      | 学习日期、主题、时长、内容、心得                 |
| Todo          | 新建/修改/删除任务、完成状态、优先级、截止日期   |
| 文件管理      | 上传/下载/删除/分类/搜索（支持 PDF/Word/Excel/PPT/Markdown/图片/ZIP） |

### 第二阶段

| 模块     | 说明                                     |
| -------- | ---------------------------------------- |
| 日记     | Markdown 编写，记录每日总结、学习心得    |
| 收藏夹   | 保存常用网址，支持分类、标签、搜索       |
| 学习计划 | 规划学习目标，记录进度、开始/完成时间    |
| 阅读记录 | 记录阅读书籍/文档，章节、进度、时间、笔记 |

### 第三阶段

| 模块     | 说明                                       |
| -------- | ------------------------------------------ |
| Dashboard | 首页系统概览，今日学习时长、待办、最近笔记 |
| 数据统计 | 学习时长、笔记/文件数量、Todo 完成率       |
| 全局搜索 | 统一搜索笔记、文件、Todo、学习记录、收藏   |
| 标签系统 | 所有模块统一标签管理                       |

### 后续可扩展

- AI 笔记总结 / AI 问答
- OCR 图片识别
- PDF 在线预览
- Markdown 导出 PDF / HTML
- 数据备份 / WebDAV 同步
- Docker 部署
- 深色模式 / 多主题切换

## 功能模块依赖关系

```
用户认证（前置）
    │
    ├── Markdown 笔记 ── 标签系统
    ├── 学习记录    ── 数据统计
    ├── Todo
    ├── 文件管理
    ├── 日记        ── 标签系统
    ├── 收藏夹      ── 标签系统
    ├── 学习计划
    └── 阅读记录
            │
            ▼
      Dashboard（聚合展示）
      全局搜索（跨模块检索）
```

## 项目原则

1. 优先保证代码可读性
2. 优先保证代码可维护性
3. 保持项目结构统一
4. 不生成重复代码
5. 优先复用已有代码
6. 不随意引入新的第三方依赖
7. 保持前后端命名一致
8. 保持接口风格统一
9. 保持数据库设计一致
10. 所有新增功能应同步更新对应文档

## 开发理念

坚持 **"边学习、边开发、边使用、边完善"** 的理念，采用模块化、渐进式开发方式，让项目随着自身需求不断成长，最终成为集知识管理、学习记录、任务管理和资源整理于一体的个人管理平台。
