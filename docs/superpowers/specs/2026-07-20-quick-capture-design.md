# 命令面板快捷捕获

> 日期：2026-07-20 · 已批准（P0 连续推进默认方案）  
> 范围：P0-C。浏览器剪藏扩展、独立速记抽屉不在本版。

## 目标

Ctrl/Cmd+K 内可直接跳转「新建」各实体，缩短捕获路径。

## 设计

| 项 | 约定 |
|----|------|
| 入口 | 现有 `CommandPalette` |
| 动作 | 与 Dashboard 快捷操作对齐：笔记 `/notes/new`；日记/阅读/学习/待办/计划走 `?create=1` |
| 空关键词 | 结果区顶部展示「新建」组（6 项），再页面路由；有搜索历史则历史仍可点 |
| 有关键词 | 按 label/别名过滤新建项，badge=`新建`，排在页面结果之前、全站搜索之后 |
| 不做 | 面板内直接写正文提交、浮动按钮、剪藏 |

## 文档

`PAGE_SPEC` / `CHANGELOG` / `PROJECT` 同步。
