# 学习计划卡片布局调整

日期：2026-07-16  
范围：仅 Product 卡片视图（`studyplan/List.vue`），表格视图与 API 不变

## 目标

将卡片信息层级调整为线框结构，分类使用软色标签，并在 5×2 网格内保持一屏可读。

## 视觉结构（自上而下）

1. **标题行**：计划名称（左）+ 状态（右；已完成为 ✅，其余为色点 + 文案）
2. **分类**：软色标签；最多显示 2 个，超出显示 `+N`（`N = tags.length - 2`）
3. **来源行**：`{icon} {来源} · {作者}`  
   - 仅有来源或仅有作者时不显示中间 `·`  
   - 皆空时显示灰色占位 `来源 —`
4. **进度**：进度条 + 百分比（沿用 `el-progress`）
5. **日期**：`YYYY.MM.DD ~ YYYY.MM.DD`；缺侧用 `—`
6. **备注**：单行省略；悬停可 tooltip 全文；无备注时保留空位以稳住卡片高度
7. **底栏**：左侧相对更新时间「更新于 …」；右侧链接图标（有 URL 时）+ `⋯` 菜单

## 相对时间规则

基于 `updatedAt`：

| 条件 | 文案 |
|------|------|
| &lt; 1 分钟 | 刚刚 |
| &lt; 1 小时 | N 分钟前 |
| &lt; 24 小时 | N 小时前 |
| &lt; 7 天 | N 天前 |
| 否则 | `YYYY-MM-DD` |

前缀统一为「更新于 」。

## 交互（不变）

- 点击卡片打开编辑
- `⋯`：编辑 / 复制 / 归档 / 删除
- 外链 `target=_blank`
- 分页 `PAGE_SIZE = 10`，网格 5×2，不足补 pad 槽位

## 非目标

- 不改表格列布局
- 不改后端字段 / API
- 不改 Dialog 表单
