# 设置 · 界面字体预览与边界强化

> 日期：2026-07-23 · 方案 B（网格 + 预览条）  
> 范围：仅系统设置「外观」中「界面字体」区块的 UI。不改阅读正文字体、不改字体预置列表与持久化。

## 目标

让用户一眼看清：① 当前选中字体长什么样；② **只影响全站 UI，不影响阅读 / Markdown 正文**。

## 背景

- `appearanceConfig.uiFont` + `uiFonts.ts` 已驱动 `--font-sans`；阅读正文走 `readingConfigStore`，二者本就不联动。
- 外观页「界面字体」现为竖排大卡片（标签 + 一行样例），与主题卡/密度卡形态不同，且边界说明仅为弱提示文案，预览不够醒目。
- 用户确认范围：强化**预览与边界**（非全设置页排版阶梯大扫除）；方案选 **网格 + 顶部 UI 预览条**。

## 设计

### 结构（自上而下）

| 层 | 约定 |
|----|------|
| 小节标题 | 仍用 `section-title`：「界面字体」 |
| UI 预览条 | 白底描边卡：左侧标签「UI 预览」；右侧徽章「阅读 / Markdown 正文不变」；中间样例「新建」按钮 + 「系统设置」+ 「工作台 · 外观」 |
| 短 hint | 一行：切换立即作用于侧栏/顶栏/按钮等壳层；正文字体请到「阅读」 |
| 字体网格 | 2 列；卡内**大样例在上、字体名在下**；5 个预置不变（思源黑/宋、霞鹜文楷、Inter、系统默认） |
| 系统默认 | 网格第 5 项通栏（`grid-column: 1 / -1`），无额外逻辑 |

### 交互

| 项 | 约定 |
|----|------|
| 点击卡片 | 立即 `setAppearance('uiFont', key)`（现有 `themeStore`）；全站 `--font-sans` 更新 |
| 预览条字体 | 仅样例文字（按钮文案、标题、路径）跟随当前选中 `fontFamily`；徽章/「UI 预览」标签保持默认 UI 字，避免边界说明难读 |
| 选中态 | accent 描边 + 浅底；字体名 accent；与主题卡 active 对齐（可含 `box-shadow: 0 0 0 1px accent`） |
| 预加载 | 保持 `onMounted` 对 `UI_FONT_OPTIONS` 预拉 stylesheet |
| 无障碍 | 卡片为 `button`；当前项 `aria-pressed="true"` |
| 窄屏 | `&lt;720px` 网格单列；预览条通栏不变 |

### 改动范围

| 路径 | 动作 |
|------|------|
| `AppearanceSettings.vue` | 重排模板与 scoped 样式 |
| `uiFonts.ts` / `themeStore` / API / 后端 | **不动**（预置与 apply 逻辑保持） |
| 阅读设置 | **不动**（仅文案指向） |

### 明确不做

- 自定义上传字体、阅读正文字体选择器
- 完整迷你壳层实时预览（方案 C）
- 全设置页字号阶梯统一（此前未选该范围）
- 改其它外观区块（主题/强调色/圆角等）

## 文档

- `PAGE_SPEC.md`：外观·界面字体 — 补「预览条 + 网格选择、边界徽章」
- `STYLE_GUIDE.md`：若有外观控件说明则补一句网格卡约定
- `CHANGELOG.md`：按日条目

## 验收

1. 外观 → 界面字体：顶部有 UI 预览条与「阅读正文不变」徽章  
2. 字体为 2 列网格，样例字明显大于字体名；选中态清晰  
3. 切换字体后侧栏/顶栏立即变；打开笔记/文档正文**不**跟换  
4. 窄屏单列可读；`uiFonts` 现有单测仍通过  
5. 无新依赖、无后端变更
