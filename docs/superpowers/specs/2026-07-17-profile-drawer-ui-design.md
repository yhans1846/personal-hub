# 个人资料抽屉 UI 重设计

日期：2026-07-17  
状态：已确认（方案 A）

## 目标

顶栏右上角头像打开的个人资料，从「左侧长表单」改为「右侧资料页感抽屉」，减少滚动、固定保存。

## 决策

| 项 | 选择 |
|----|------|
| 滑出方向 | 右侧（`direction="rtl"`） |
| 分区 | 分段 Tab：基本 / 联系 / 简介 |
| 顶部 | Hero：头像 + 昵称 / `@用户名` |
| 底部 | 吸底：重置 / 保存 |
| API / 字段 | 不变 |

## 结构

```
[标题 个人资料]                    [关闭]
[ Hero: 头像 + 昵称 / @username ]
[ 基本 | 联系 | 简介 ]   ← segment
[ 当前 Tab 表单字段 … ]
────────────────────────
[ 重置 ]              [ 保存 ]
```

## 字段归属

- **基本**：用户名（只读）、昵称、性别、出生日期
- **联系**：邮箱、手机号、所在地区
- **简介**：GitHub、个人网站、个人简介

## 非目标

- 不改后端接口与校验规则
- 不迁入设置页；仍为独立抽屉
