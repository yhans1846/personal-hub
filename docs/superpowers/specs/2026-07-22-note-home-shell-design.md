# 笔记模块「首页」与语雀式双栏壳

> 日期：2026-07-22 · 已确认（语雀式 A · 树含笔记 · 保留卡片/表格 · 笔记壳双栏 A）  
> 前置：`2026-07-22-note-folder-tree-design.md`（文件夹 CRUD/拖拽/计数仍有效，本规格扩展导航与默认页）  
> 修订：2026-07-23 — 编辑 Overlay / 预览页亦展示库树（见文件夹树规格修订）。

## 1. 目标

- 进入 `/notes` **默认右侧为知识库首页**（非卡片列表）。
- 左侧为笔记模块内导航：首页入口 + 文件夹树（**夹下挂笔记**）。
- 点文件夹 / 未分类 → 右侧仍为**现有卡片/表格列表**（多选、导入导出等能力保留）。
- 点树中笔记 → 现有 Overlay 编辑。

成功标准：

1. 打开笔记菜单默认看到首页（标题、文档数、欢迎区、带更新时间的目录列表）。
2. 侧栏可展开文件夹看到其下笔记并点击打开。
3. 选中夹后右侧列表筛选行为与现文件夹树一致；工具栏能力不丢。
4. 编辑 Overlay / 预览页与列表共用库树（点笔记切换；点夹仅改选中）。

## 2. 信息架构

| 区域 | 行为 |
|------|------|
| App 主侧栏 | 不变；「笔记」仍进 `/notes` |
| 笔记内左栏 | 「首页」+ 文件夹树（夹+笔记）+ 未分类；≤768 抽屉 |
| 右侧 · 首页模式 | 库概览 + 目录列表（夹可展开，笔记行显示更新时间） |
| 右侧 · 列表模式 | 现 `List` 卡片/表格（由选中夹/未分类/全部驱动） |
| 树 · 笔记节点 | 点击 → Overlay `edit` |
| 树 · 文件夹节点 | 点击 → 右侧切列表并筛选该夹直属笔记 |

明确不做（本期）：独立「目录」路由页、多知识库、分享/收藏顶栏、字数统计、三栏 App+树+内容。

## 3. 布局（笔记壳双栏）

在现有 `plan-page` / `List.vue` 上演进为：

```
┌─────────────┬──────────────────────────────┐
│ 首页         │  Home  或  ListToolbar+列表   │
│ 未分类       │                              │
│ ▾ 夹         │                              │
│   笔记标题   │                              │
└─────────────┴──────────────────────────────┘
```

- 路由仍以 `/notes` 为主；模式用本地状态（`viewMode: 'home' | 'list'`），可用 query `?folder=` / `?home=1` 可选增强（非必须）。
- 默认：`viewMode = 'home'`；点夹/未分类 → `list`；点「首页」→ `home`。

## 4. 首页内容

| 块 | 内容 |
|----|------|
| 头图区 | 图标 + 标题「笔记」+「N 文档」（N = 未删除笔记总数，复用树接口 `totalCount`） |
| 欢迎区 | 简短说明文案（固定文案即可） |
| 目录区 | 按文件夹树展开：夹名一行；其下直属笔记：标题 …… 更新时间（点笔记开编辑） |

字数统计、收藏/分享按钮：**不做**。

## 5. 侧栏树扩展

在现有 `NoteFolderTree` 基础上：

- 每个文件夹节点可展开显示**直属笔记**（标题；可选截断）。
- 数据：扩展 `GET /api/note-folders/tree`（或并列接口）返回每夹下笔记摘要列表；未分类笔记挂在「未分类」下。
- 笔记节点：不可拖成夹的子「夹」；笔记拖入夹的能力可保留（拖到夹行 = 改 folderId）。
- 夹的 CRUD / 平级与嵌套拖拽 / 计数：**保持**现有行为。

建议节点摘要字段：`{ id, title, updatedAt, folderId }`。

## 6. API

| 项 | 约定 |
|----|------|
| 树 | 现 `NoteFolderTreeVO` 扩展：节点增加 `notes?: NoteFolderNoteItem[]`；根级增加 `uncategorizedNotes`（或未分类作为虚拟节点由前端拼） |
| 首页 | 可不另开接口：用 tree + totalCount 渲染目录；若笔记列表过大，可限每夹最近 N 条并提供「在列表中查看」 |

一期优先：**一次 tree 带齐夹下笔记摘要**（仅 id/title/updatedAt，排除回收站）。个人库规模可接受。

## 7. 前端落点

| 文件 | 职责 |
|------|------|
| `List.vue`（或拆 `NoteShell.vue`） | 双栏壳；`home \| list` 切换；托管树 + Overlay |
| `NoteFolderShell.vue` | 列表/Overlay/预览共用左树壳（收起轨、移动端抽屉） |
| `NoteHome.vue` | 首页头图/欢迎/目录列表 |
| `NoteFolderTree*.vue` | 渲染笔记子节点；选中首页/夹/笔记；`activeNoteId` 高亮当前 |
| `NoteWorkspaceOverlay.vue` / `Preview.vue` | 左树 + 主区；树点笔记切换 |
| `api.ts` / types | 树 VO 扩展 |

## 8. 与旧规格关系

- `2026-07-22-note-folder-tree-design`：删夹策略、拖拽、直属筛选、备份等**仍然有效**。
