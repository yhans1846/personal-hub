# UI 设计方案（UI Design）

> 更新时间：2026-07
>
> 本文档用于规范 Personal Hub 项目的整体 UI 风格、页面布局、交互设计及后续优化方向，确保整个项目保持统一的视觉体验。

---

# 一、设计理念

## 设计关键词

- 简洁（Minimal）
- 专注（Focus）
- 温暖（Warm）
- 高级（Premium）
- 长期使用（Long-term）

Personal Hub 不是企业后台，而是一个长期陪伴自己的个人知识管理系统，因此页面应该：

- 不追求炫酷动画
- 不追求复杂视觉
- 保持极低学习成本
- 保持舒适阅读体验
- 长时间使用不会疲劳

整体风格参考：

- Notion
- Linear
- Raycast
- Arc Browser
- Apple Human Interface

而不是：

- 大屏风
- 企业 ERP
- OA 系统
- 传统 Admin Template

# 三、页面宽度

## 当前问题

目前内容区域过窄。

例如：

```
□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□

       ████████████

□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□
```

右侧大量留白，视觉不平衡。

---

## 推荐方案

内容区域最大宽度：

```
1400px
```

或者

```
1600px
```

推荐：

```css
width: calc(100vw - 260px);
max-width: 1600px;
```

让内容自然铺满屏幕。

# 十八、最终目标

打造一个：

- 不像企业后台
- 不像 Admin Template
- 更像现代桌面应用
- 长时间使用不疲劳
- 专属于自己的知识管理中心

希望最终达到：

> **Notion 的简洁 + Linear 的精致 + Raycast 的现代感 + Apple 的克制设计。**
