# 一、设计目标

系统设置（Settings）不仅是功能配置页面，更是整个 Personal Hub 的「工作台配置中心（Workspace Preferences）」。

设计目标：

- 简洁
- 高级
- 可扩展
- 一致
- 所见即所得
- 自动保存

参考产品：

- Notion
- Obsidian
- VS Code
- Raycast
- Arc Browser
- Vercel Dashboard

---
